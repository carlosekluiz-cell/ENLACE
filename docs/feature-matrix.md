## Notas

1. Todos os tiers incluem autenticacao JWT e dados atualizados automaticamente pelos 38 pipelines.
2. O tier Gratuito e projetado para demonstrar valor e converter para tiers pagos.
3. Precos em BRL, cobrados mensalmente (Provedor) ou anualmente com desconto de 15% (Profissional e Empresa).
4. API rate limits: Profissional = 100 req/min, Empresa = sem limite.
5. Dados de base: todos os tiers acessam os mesmos dados reais (28M+ registros). A diferenca esta na profundidade de analise e ferramentas disponiveis.
